# SCUMMKit v1.0.0

First release of SCUMMKit.

Highlights:
- Native Monkey Island 1 Ultimate Talkie Edition builder
- Native Monkey Island 2 Ultimate Talkie Edition builder
- Python CLI: `python3 -m scummkit`
- Native PAK extraction
- Native XWB extraction
- Native speech archive generation
- Native replacement for `build_monster.exe`
- Native replacement for `wav2sbl.exe`
- Native replacement for `scummpacker.exe` SBL injection path
- MI1 music conversion using `vgmstream-cli`
- MI1 soundtrack selection: `--music cd|hybrid|se`
- Resource inspection commands
- Pytest test suite

Supported games:
- The Secret of Monkey Island
- Monkey Island 2: LeChuck's Revenge

Legal note:
- No game assets are included.
- Users must provide legally owned game files and the Ultimate Talkie builder files.
